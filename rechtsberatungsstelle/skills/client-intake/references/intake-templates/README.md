# Mandatsaufnahme-Vorlagen nach Rechtsgebiet

Wird bei der Ersteinrichtung mit den Aufnahmeformularen der Lehrperson befüllt. Liegen keine eigenen Formulare vor, verwendet `/mandatsaufnahme` die Standard-Fragekataloge aus `mandatsaufnahme/SKILL.md` Schritt 2.

Ein Dokument pro Rechtsgebiet, das die Beratungsstelle bearbeitet:

- `einwanderungsrecht.md` — Aufenthaltsstatus, Einreise, frühere Anträge, Ländersituation, Familie, Vorstrafen, Zeitverlauf
- `mietrecht.md` — Wohnverhältnis, Sachverhalt, Mietvertrag/Zahlung, Wohnungsmängel, Zeitverlauf
- `familienrecht.md` — Beziehungsverhältnis, Kinder, Sicherheitslage, bestehende gerichtliche Anordnungen, Zeitverlauf
- `verbraucherrecht.md` — Art der Forderung, wer Kontakt aufnimmt, Unterlagen, gerichtliche Verfahren, Zeitverlauf

Jede Vorlage enthält die zu stellenden Fragen in der empfohlenen Reihenfolge sowie Hinweise zum sensiblen Umgang (z. B. Vorstrafen im Einwanderungsrecht — erläutern, warum diese Frage gestellt wird).
