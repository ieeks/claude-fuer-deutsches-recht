# Testakte Energierecht: Stadtwerke-Quartier und Industriekunde

Fiktive Testakte für ein Stadtwerk, das ein neues Quartier mit Wärme, PV-Strom, Ladepunkten und einem Industriekundenanschluss strukturiert. Enthalten sind Intake, Projektdaten, Netzanschluss, Wärmevertrag, Industriekunden-Umlagen, Transaktions-DD und Verfahrenspost.
